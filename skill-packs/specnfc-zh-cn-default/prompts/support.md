# 辅助 Prompt 目录

## 使用目的
- 为中文 `nfc` skill-pack 提供稳定的角色 / 阶段 / 辅助提示词目录。
- 提示词服务于当前阶段推进，不得自建独立流程真相。

## 目录条目
- 文档规范化：按 document contract 收口文档。
- 决策提炼：把讨论结果提炼进 decisions.md。
- 会议记录转 dossier：把过程记录回写到正式 dossier。
- writeback：把运行时内容写回正式文档并更新队列。
- 下一步建议：统一生成标准 next-step 输出。

## 共通输出要求
- 当前阶段
- 已完成
- 缺失 / 阻断
- 推荐下一步
- 是否需要 writeback
- 是否存在 projection / skill-pack drift
