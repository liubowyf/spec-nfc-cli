# 阶段 Prompt 目录

## 使用目的
- 为中文 `nfc` skill-pack 提供稳定的角色 / 阶段 / 辅助提示词目录。
- 提示词服务于当前阶段推进，不得自建独立流程真相。

## 目录条目
- 需求澄清：统一问题定义、范围、非目标、验收口径。
- 方案设计：收敛结构、约束、影响面、替代方案与验证思路。
- 任务规划：拆任务、定验证、定依赖。
- 执行落地：按 plan 实施并持续回填状态。
- 系统化调试：先复现、后定位、再修复，保留证据链。
- 评审复核：形成结构化 review 结论与阻断摘要。
- 验证验收：形成结果、证据、剩余风险与 accept 结论。
- 发布交接：形成交付、回退、验证与接手信息。

## 共通输出要求
- 当前阶段
- 已完成
- 缺失 / 阻断
- 推荐下一步
- 是否需要 writeback
- 是否存在 projection / skill-pack drift
