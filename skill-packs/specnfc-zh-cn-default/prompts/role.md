# 角色 Prompt 目录

## 使用目的
- 为中文 `nfc` skill-pack 提供稳定的角色 / 阶段 / 辅助提示词目录。
- 提示词服务于当前阶段推进，不得自建独立流程真相。

## 目录条目
- 架构师：聚焦边界、约束、接口和长期可维护性。
- 规划师：聚焦阶段拆分、依赖顺序、验证计划。
- 执行者：聚焦最小正确改动、严格验证和状态回填。
- 调试者：聚焦复现、根因、失败路径和修复验证。
- 评审者：聚焦设计/实现/对接结论、阻断项和建议项。
- 验证者：聚焦证据、验收结论、剩余风险。
- 风险审阅者：聚焦阻断、豁免、合规边界和回退。
- 交付整理者：聚焦 handoff、发布说明和归档收口。

## 共通输出要求
- 当前阶段
- 已完成
- 缺失 / 阻断
- 推荐下一步
- 是否需要 writeback
- 是否存在 projection / skill-pack drift
