# `Spec nfc` 总览

`Spec nfc` 用来组织业务仓里的规格、变更和工具入口。

当前提供：
- 按模式初始化：`minimal / standard / enterprise`
- 变更生命周期：`create / list / check / stage / handoff / archive`
- 分层变更骨架：`proposal / design / spec / capabilities / spec-deltas / plan / tasks / decisions / status`
- 详细的上下文、执行、治理、接口设计、数据库设计和质量模板
- `change` 目录与 delivery 文件联动
- `Codex / Claude Code / OpenCode / Trae` 多工具统一入口
- 个人 skills 与仓级规范的隔离规则
- 团队运行说明入口：`specnfc explain runtime`
- 跨平台一键安装脚本
